# Back To Basics

BTB is a shader that emulates the looks of 3D PS1 games.

**Features:**

- Low screen res
- Vertex snapping
- Affine texture mapping
- 16bit color overall
- Dithering
- Configs
- Distant Horizons Support

**Version:** 2.0.0

Check it on [Modrinth](https://modrinth.com/shader/btb-back-to-basics)
